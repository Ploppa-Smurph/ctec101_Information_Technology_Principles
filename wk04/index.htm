- Desktop - often used to connect to a network and can be considered a network client.
- All-in-one - more difficult to upgrade and repair, normally no expandable memory or internal storage
- Laptop or Notebook - monitor, keyboard, mouse, and networking are built together with cpu for portability. more expensive than comparable desktop computers, more difficult to upgrade and repair
- Hybrid-Laptop Device - 2-in-1 devices cater to need of tablet and laptop user.
- Gaming Console - built for gaming and use with gaming controller instead of keyboard and mouse.
- Network Client - a device that can connect to a network -- all of these devices listed can be network clients.
- Mobile - often use touchscreen. -- less computing power than other devices.
- Servers - purpose is to share disk storage or perform service for large number of people, such as web, email, or database services.

3.3.3 - Electrostatic Discharge

ESD Prevention -- 1st discharge yourself before starting -- touch the case
2nd - use static mats and static floor mats to keep the charge from building up

4th - keep components in the static shielding bag (heavy grey bag that computer parts come in)
5th - keep plastic and styrofoam away from the item you are working on
6th - humidity is better --- because dry air builds up electrostatic charge more quickly

3.3.7 - Motherboards and Expansion Cards

the BUS - short for omnibus ('for all') allows for communication across components on a MotherBoard - the Bus is all the circuitry on the MotherBoard
the Bus is responsible for transferring the following:
  • Power- bus provides the electricity from the power supply to each component
  • Data- information transferred between the components in the computer.
  • Memory Location- bus keeps track of the address where the data is stored in the memory
  • Clock Timing- has a clock mechanism which synchronizes how all components work together over the bus


    • the bus architecture has changed over the years.
  • PCI - Peripheral Component Interconnect - usually 32-bits wide, although 64-bit versions were used as well. runs @ 33mhz can transfer data @ 133 MB/s (or 266 MB/s for 64-bit version)
  • PCI with AGP - similar to PCI but specifically for graphics support. replaced with PCIe in modern systems
  • PCI Express (PCIe) - next gen I/O bus architecture. each PCIe slot links to a switch which prioritizes the routes of data through a point-to-point dedicated connection and provides a serial full-duplex method of transmission.
    • basic PCIe provides one lane for transmission at a rate of 250 MB/s per lane for V1. V2 can deliver 500 MB/s; V3 = 1 GB/s; and V4 = 2 GB/s per lane. PCIexpress slots can have 2x, 4x, 8x, 16x, & 32x transmission lanes so a V4 with 32 lanes of transmission would be super fast
    • in addition to increased speed, PCIe offers higher quality service
    • PCIe is backwards compatible and allows legacy PCI tech to be run in the same system
    • PCIe busses are most commonly used for video cards in modern systems, although any device can be designed for a PCIe slot

    • Expansion Cards are circuit boards that connect to the computer's MoBo - the expansion card gives the MoBo additional power or functionality.

    video, network, and audio expansion cards are the most common type, although additional usb, or media reader slots would be types of expansion cards, also
    network cards feature RJ-45 ports to connect a computer, via unshielded twisted pair cable, to the network. aka ethernet ports, and now wifi connectivity.

    Audio cards feature a DSP chip (Digital Signal Processor) - DSP chips are optimized for converting electrical signal to audio signal.

                            Troubleshooting Memory Errors
One reason you might experience memory errors is that the memory card isn't properly installed or has come loose. Also, the socket that the memory card is seated in might have gone bad or needs cleaning. The following steps can help you diagnose and fix the issue.

1.Make sure that the memory card is compatible for the computer, especially if you're getting errors after installing new memory on the system. Compatibility specs can be found on the manufacturer's website.

2.Confirm that you've configured the memory correctly. The correct configuration depends on the kind of computer you're using. The documentation that comes with the computer, the manufacturer's website, or the the manufacturer's technical support are all good resources to find the configuration the computer requires.

3.If everything is installed correctly, try re-installing the memory module. Push the module firmly into the socket until you hear a click. Then, compare the height of that module to the surrounding modules to make sure it has a similar height.

4.If the above steps don't fix the problem, try removing any newly installed memory and see if the problem persists. You can also try swapping the memory into different slots. If the memory works in one socket but not in another, the problem may be with the socket and not the memory module.

5.If a socket isn't working, you can clean the socket to see if that improves the situation. First, use a gentle cloth and wipe the pins on the module. You can then use compressed air to blow any built-up dust from the socket. There are also special kinds of cleaners that you can purchase at computer equipment stores. Do not use a solvent as this can cause further damage to the socket.

6.Finally, try to update the BIOS information, especially if you've recently upgraded the memory. The BIOS is regularly updated by motherboard manufacturers. These updates can help to fix bugs that crop up during the life cycle of a particular motherboard. If the BIOS is not up to date, it can contribute to memory problems.

7.If none of these steps work, you may have faulty hardware. If the computer is still under warranty, contact the manufacturer to schedule a repair or replacement.
